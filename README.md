# Supplementary-material
Innovation and Individual Consistency in Bottle Manipulation Behaviour of Wild Chacma Baboons (Papio ursinus)

This contains all supplementary files associated with the research project.

Raw bottle data.xlsx - Data for each bottle manipulation event opening/drinking used for analyis
Raw ranking data.xlsx - Data of winners and losers from interactions used to calculate dominace rank

S Female Rank.R - R code used to calculate and plot rank of Sehlahla females
S Male Rank.R - R code used to calculate and plot rank of Sehlahla males
T female Rank.R - R code used to calculate and plot rank of Toropo females
T Male Rank.R - R code used to calculate and plot rank of Toropo males
Drinking models and barplots.R - R code used to run models associated with drinking methods, and plot barplots of probability of method used by each individual for both drinking and opening
Opening models and Conspicousness model.R - R code used to run models associated with opening methods, model of relationship between rank and conspicousness, and plot linegraph of rank/probability 

Over head.mov - Video showing drinking over head method
Pour on floor.mov - Video showing pouring liquid on floor and licking method
Rip lid with teeth.mov - Video showing ripping lid with teeth method
